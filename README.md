# 프로필 화면

- 사용자 프로필 정보 표시
    - 게시물, 팔로워, 팔로잉 수
- 피드 리스트 표시
    - 게시물, 좋아요, 즐겨찾기 피드 리스트

# 사용 기술

| 종류           | 기술                             |
|--------------|--------------------------------|
| Language     | Kotlin                         |
| UI           | Jetpack Compose                |
| 모듈화          | Jitpack 플랫폼 사용                 |
| Architecture | AAC (UI - Domain - Data Later) |
